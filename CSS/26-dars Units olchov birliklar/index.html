Sarlavha 1

Absolute -> O'lchov birligi

Absolute -> (mutloq) units - mutloq o'lchov birliklari qanday korsatilgan bolsa huddi shunday ekranga chiqadi. Absolute o'lchov birliklarini ishlatish tavsiya etilmaydi chunki ular ekranga qarab oz korinishini o'zgartirmaydi

Absolute qiymatlar

px

cm

mm

in

pt

Relative -> O'lchov birligi

Relative -> (nisbiy) units - nisbiy olchov birliklari boshqa uzunlik o'lchoviga nisbatan o'z uzunligini belgilaydi.Turli xil ekranlarga qarab moslashadi.

Relative qiymatlar

%

em

rem

vw va vh (viewport units)
